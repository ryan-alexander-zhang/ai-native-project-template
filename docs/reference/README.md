# References

This directory stores external reference materials.

## Naming

- filenames should start with a datetime in `YYYYMMDDHHMMSS` format, followed by `-<slug>.md`
- example: `20260525202015-api-provider-docs-summary.md`

## Must Include

- external links
- paper or article summaries
- official doc excerpts
- benchmark research

Add more when useful.

## Exclude

- final project decisions
- final rules and constraints

## Note

References are for input only. They do not replace formal documents.
