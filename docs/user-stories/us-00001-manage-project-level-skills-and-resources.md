# User Story

As an independent developer or small team lead, I want to add, inspect, and update project-level skills and reusable resources stored with the project so the repository keeps its AI workflow setup in one place.

# Acceptance Criteria  
- [ ] AC-01: A user can add a project-level skill or reusable resource to the project and see it persisted with the project.
- [ ] AC-02: A user can inspect the stored project-level skills and reusable resources from the project.
- [ ] AC-03: A user can update an existing project-level skill or reusable resource and see the saved version reflected on reload.

# Definition of Done  
- [ ] The project-scoped skills and resources workflow supports add, inspect, and update operations.
- [ ] The supported skills and resources behavior is documented for repository users.
- [ ] Tests or validation cover add, inspect, and update behavior.
