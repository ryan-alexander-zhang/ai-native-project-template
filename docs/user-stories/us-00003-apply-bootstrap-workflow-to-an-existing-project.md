# User Story

As an independent developer, small engineering team member, or technical lead, I want to run the bootstrap workflow against an existing local project so that managed resources can be added or standardized in place.

# Acceptance Criteria  
- [ ] AC-01: Given an existing local project, when the bootstrap workflow is run, then it applies the managed resource setup to that project.
- [ ] AC-02: Given managed resources already exist, when the workflow runs, then it standardizes them instead of creating a separate project scaffold.
- [ ] AC-03: Given the workflow completes, then the user can see which resources were added or standardized.

# Definition of Done  
- [ ] The story is implemented for existing local projects.
- [ ] The workflow outcome is verifiable in the repository.
- [ ] The change is covered by tests or a documented validation step.
